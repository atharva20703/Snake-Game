# Snake Game Using Python and Tkinter

## Overview
This project is a classic Snake Game built using Python and the Tkinter library for the graphical interface. The player controls a snake, which grows in size as it eats food. The goal is to avoid collisions with the snakeâ€™s own body and the edges of the game window.

## Features
- Real-time snake movement controlled by arrow keys.
- Dynamic food spawning.
- Score tracking system.
- Game over screen when the snake collides with the walls or its own body.
